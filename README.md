# QiQ v1 – Quick Insurance Quotes for Ghana

QiQ is a lightweight insurance‑quote engine designed for the Ghanaian market.  
It includes:

- Chat‑based quote flow (Auto, Home, Life, Health)
- Curated insurance product dataset
- Dashboard with analytics + trends
- Broker vs Insurer comparison view
- Public API with rate limiting
- Nightly GitHub Action scraper pipeline
- Netlify serverless backend

---

## 🚀 Live Features

### **1. Chat Quote Engine**
Guides users through a simple conversation and returns estimated premiums.
